# 🎯 COMECE AQUI - Coloque Seu Site Online em 10 Minutos!

## ⚡ Resumo Rápido

Seu site **Forjado na Rocha** está **100% pronto** para ir online! Siga estes 3 passos simples:

### **Passo 1: Criar Conta GitHub (2 minutos)**
1. Acesse [github.com](https://github.com)
2. Clique em **"Sign up"**
3. Preencha o formulário e confirme seu email
4. Crie um novo repositório chamado **"forjado-na-rocha-web"**

### **Passo 2: Fazer Push do Código (3 minutos)**
Execute este comando no seu computador:

```bash
cd /home/ubuntu/forjado-na-rocha-web
git remote add origin https://github.com/SEU_USUARIO/forjado-na-rocha-web.git
git branch -M main
git push -u origin main
```

**Substitua `SEU_USUARIO` pelo seu usuário GitHub**

### **Passo 3: Deploy no Vercel (5 minutos)**
1. Acesse [vercel.com](https://vercel.com)
2. Clique em **"Sign Up"** → **"Continue with GitHub"**
3. Autorize o Vercel a acessar seu GitHub
4. Clique em **"New Project"**
5. Selecione **"forjado-na-rocha-web"**
6. Clique em **"Deploy"**
7. **PRONTO!** Aguarde 1-2 minutos

## 🆘 Precisa de Ajuda?

### Erro: "Cannot GET /"
- Verifique se `public/index.html` existe
- Verifique os logs no Vercel

### Erro ao fazer push
- Verifique sua conexão com internet
- Confirme que o repositório existe no GitHub
- Verifique se você tem permissão

### Site está lento
- Verifique os logs no Vercel
- Use o CDN do Vercel (já incluído)

### Domínio não funciona
- Aguarde 24-48 horas para propagação DNS
- Verifique se os DNS records estão corretos
